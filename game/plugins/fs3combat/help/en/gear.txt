There are various kinds of gear that can be used in combat.   For more details on FS3, see the game wiki or %xhhelp combat%xn.

    %xcweapons%xn - List all weapons.
    %xcweapon <name>%xn - See details for a particular weapon.
    
    %xcarmor%xn - List all types of armor.
    %xcarmor <name>%xn - See details for a particular armor type.
    
    %xcvehicles%xn - List all types of vehicles.
    %xcvehicle <name>%xn - See details for a particular vehicle type.
