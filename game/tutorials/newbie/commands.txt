All the things you've been typing so far - 'connect', 'tutorial', 'who' - are %xhcommands%xn.  There are a ton of other commands out there to support your roleplay.

One very important command is the %xhhelp%xn command, because this will give you information about all the other commands and how to use them.   To see a list of help topics, just type:

    %xchelp%xn

Note: Throughout Ares help files, the light blue color is used to indicate commands you type.

You can read a specific help file by typing:

    %xchelp <name>%xn  --- where <name> is the name of the help file.

%xh-- Give it a try! --%xn 
    Type: %xchelp%xn and %xchelp tutorials%xn
